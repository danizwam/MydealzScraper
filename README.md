# MydealzScraper
Scraper for mydealz.de. Not affiliated with MyDealz in any way. Uses parts from [mydealz_alert](https://github.com/pfannkuchengesicht/mydealz_alert/).

## Requirements
Install the following libraries to use this script:
* bs4
* colorama
